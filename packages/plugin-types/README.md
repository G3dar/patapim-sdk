# @patapim/plugin-types

Reserved for **Phase 2: the PATAPIM plugin system** — `plugin.json` manifest schema and host API type definitions.

Plugins will live in `~/.patapim/plugins/<name>/`, declare `permissions` (the same scopes the Local API uses today) and `contributes` (toolbar buttons, commands, panels, MCP tools, instruction blocks, scheduled tasks), and run out-of-process with a scoped token.

Nothing to install yet — watch the repo for the announcement.
